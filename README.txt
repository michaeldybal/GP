Реализовать программу на Java:
- на вход которой подается строка с цифрами, знаками +-*/ и скобками;
- на выходе результат вычисления.

Использовать принципы ООП:
- предусмотреть возможность чтения входных данных из разных источников:
консоль/БД/файл (реализовать минимум из одного из источников)
- предусмотреть возможность разного вывода данных: консоль/БД/файл/UI
(реализовать минимум в один из источников)
- предусмотреть возможность подключения разных алгоритмов вычисления
(имеется в виду работа с 2-ым, 10-ым, 16-ым форматами ввода исходных данных)
- * использовать встроенную в jdk СУБД derby, в которой завести таблицу с 2
столбцами: входная строка и результат. Записывать в нее данные.
- * реализовать данный функционал как одностраничное web-приложения
(используя сервлет и, например, JSP/JSF страницы). Запуск на tomcat.
